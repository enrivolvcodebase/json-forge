# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-01

### Added
- Initial release of @enrivol/json-forge
- JSON string to object parsing
- Object to JSON string conversion
- Automatic TypeScript interface generation
- Automatic JavaScript JSDoc type generation
- Configurable model output path
- Configurable interface/type naming
- JSON prettification option
- Full TypeScript support
- Comprehensive test suite
- Complete documentation and examples

### Features
- Zero runtime dependencies
- Type-safe API
- Flexible configuration options
- Support for nested object types
- Array type inference
- Export control for generated models
